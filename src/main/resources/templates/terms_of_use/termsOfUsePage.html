Postanowienia ogólne

Rejestrując się w systemie wyrażasz zgodę na upublicznienie Twoich aktywności (ocenianie i komentowanie filmów) innym użytkownikom systemu.

Źródło danych

Źródłem danych jest portal TMDb. Ten produkt wykorzystuje interfejs API TMDb, ale nie jest zatwierdzony ani certyfikowany przez TMDb.

Komunikacja z API TMDb odbywa się przy pomocy polecanej przez portal biblioteki Java.
